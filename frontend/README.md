# CVNHS Electronic Research Library

## Project info

This is a Vite + React + TypeScript application using shadcn-ui and Tailwind CSS.

## Getting Started

To get started with local development:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
